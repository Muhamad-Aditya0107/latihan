Pendahuluan JavaScript

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. JavaScript dapat dijalankan di browser web dan server melalui Node.js.

Sintaks Dasar JavaScript

Sintaks Dasar

JavaScript sensitif terhadap huruf besar-kecil (case-sensitive) dan setiap perintah diakhiri dengan tanda titik koma (;). Penulisan komentar bisa menggunakan "//" untuk komentar satu baris atau "/* ... */" untuk komentar beberapa baris.

Variabel

Variabel dalam JavaScript

Variabel digunakan untuk menyimpan data. JavaScript menggunakan kata kunci var, let, dan const untuk mendeklarasikan variabel.

  • var: Dapat digunakan secara global atau lokal.
  • let: Dapat digunakan dalam lingkup lokal (block-scope).
  • const: Variabel konstan yang nilainya tidak dapat diubah.

Tipe Data

Jenis Tipe Data

Tipe data JavaScript termasuk Number, String, Boolean, Array, Object, dan null.

Operator

Operator dalam JavaScript

JavaScript memiliki beberapa jenis operator:

  • Aritmatika: +, -, *, /, %
  • Perbandingan: ==, ===, !=, !==, >, <, >=, <=
  • Logika: &&, ||, !
  • Penugasan: =, +=, -=, *=, /=

Fungsi

Mendeklarasikan Fungsi

Fungsi adalah blok kode yang dapat dipanggil untuk menjalankan tugas tertentu. Fungsi dideklarasikan menggunakan function diikuti dengan nama fungsi.

function namaFungsi() { 
    // kode di sini 
}

Objek

Objek dalam JavaScript

Objek adalah koleksi properti dan metode. Properti adalah nilai yang terkait dengan objek, sedangkan metode adalah fungsi yang terkait dengan objek.

Event

Event dalam JavaScript

Event adalah tindakan yang terjadi ketika pengguna berinteraksi dengan halaman web, seperti klik, hover, submit, dll. Event listener digunakan untuk menangani event ini.

DOM (Document Object Model)

Mengelola DOM

DOM adalah representasi struktural dari dokumen HTML. Dengan JavaScript, kita bisa mengakses dan memodifikasi elemen-elemen HTML melalui DOM.

Validasi Formulir

Validasi Formulir dengan JavaScript

JavaScript dapat digunakan untuk memvalidasi data formulir sebelum dikirim ke server. Validasi ini memastikan data yang dimasukkan sudah benar dan sesuai format.
